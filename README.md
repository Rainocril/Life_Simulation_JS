# Симуляция жизни с помощью нейросетей, методом обучения с подкреплением.

## Краткие возможности (Управление)
### Дебаг работы с гексагонами 
(включение через '~' и последующей перемоткой через 'D') (да-да, сделано криво):
* Определение выбранного гексагона.
* Проверка работы LoS (линия взгляда/недорейтрейсинг).
### Дебаг зоны видимости существ 
(переключается нажатием 'V' и сменой чисел '1-3' в зависимости от типа существа):
* Зона видимости (зона которую видно).
* Работа LoS для определения зоны видимости.
* Радиус видимости и проверка сортировки гексагонов (переключение стрелочками верх-вниз).
* Определение дальности от центра.
### Смена режима симуляции
(переключение через 'E')
* симуляция на полномасштабной карте со спавном всех существ.
* тестовая симуляция со спавном одного конкретного сущесетва на маленьком участке карты.
### Регенерация карты
(нажатие на 'Space')
* Генерация новой местности.
### Пауза
('P')
### Смена разрешения карты (качества карты)
(интерактивные кнопки в левом нижнем углу)
### Запуск/выключение симуляции
(нажатие на 'Enter')

## Краткое описание
Симуляция жизни на гексогональной двумерной сетке, генерирующейся через шум. Попытка реализации разнообразной модели хищник-жертва с использованием нейронных сетей метода обучения с подкреплением.
